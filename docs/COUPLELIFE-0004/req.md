# COUPLELIFE-0004 專屬設定頁面開發 需求規格

## 功能描述

建立「設定」頁面，提供兩個功能：
1. **伴侶連結邀請碼**：顯示邀請碼並提供複製功能
2. **版本號碼**：顯示當前 APP 版本（從 BuildConfig.VERSION_NAME 取得），格式 `v{versionName}`

---

## 使用者情境

**誰**：APP 使用者  
**情況**：進入設定頁籤  
**發生什麼**：想查看邀請碼或確認 APP 版本  
**期望結果**：清楚看到邀請碼（可複製）與版本號碼

---

## 驗收條件

- [x] 進入頁面顯示伴侶連結邀請碼區塊
- [x] 邀請碼旁有複製按鈕，點擊後複製至剪貼簿並顯示提示
- [x] 頁面底部顯示版本號碼，格式為 `v{versionName}`（如 `v1.0`）
- [x] 版本號碼從 BuildConfig.VERSION_NAME 動態取得，不寫死

---

## 邊界條件

- **邀請碼**：此階段為靜態展示（hardcode 或 DataStore 儲存），不需後端 API
- **版本格式**：直接顯示 `v` + versionName，不額外格式化
- **複製提示**：使用 Toast 或 Snackbar 顯示「已複製邀請碼」

---

## 不在範圍內（Out of Scope）

- 邀請碼的後端生成 / 驗證
- 伴侶配對邏輯
- 帳號登入 / 登出
- 其他設定項目
