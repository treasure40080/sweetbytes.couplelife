# COUPLELIFE-0001 APP初次開發 — 我們的家頁面 需求規格

## 功能描述

APP 首次開發，建立「我們的家」首頁頁面，進入頁面後顯示一個滿版 Lottie 動畫作為主視覺。

---

## 使用者情境

**誰**：所有 APP 使用者  
**情況**：開啟 APP 後進入首頁  
**發生什麼**：目前尚無任何頁面  
**期望結果**：看到一個滿版 Lottie 動畫佔滿整個螢幕

---

## 驗收條件

- [x] 「我們的家」頁面存在並可正常進入
- [x] Lottie 動畫滿版顯示（寬高佔滿整個螢幕）
- [x] 動畫可正常播放（不閃退、不空白）

---

## 邊界條件

- **動畫檔案**：Lottie JSON 檔案需放置於 assets 目錄
- **滿版定義**：fillMaxSize()，不留任何 padding/margin

---

## 不在範圍內（Out of Scope）

- 動畫播放完畢後的後續互動行為
- 使用者登入 / 配對邏輯
- 其他頁面開發
