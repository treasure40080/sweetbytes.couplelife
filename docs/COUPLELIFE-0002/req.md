# COUPLELIFE-0002 記帳流水頁面開發 需求規格

## 功能描述

建立「記帳流水」頁面，提供雙人記帳功能。使用者可新增收支記帳項目，並依年/月篩選瀏覽歷史紀錄。資料持久化儲存於本地端（Room + DataStore），重啟 APP 後仍可看到歷史資料。

---

## 使用者情境

**誰**：APP 使用者  
**情況**：進入記帳流水頁面  
**發生什麼**：需要新增或查看收支記帳紀錄  
**期望結果**：可新增記帳、切換月份瀏覽、資料重開 APP 後仍存在

---

## 驗收條件

- [x] 進入頁面預設顯示當月記帳清單
- [x] 可切換年/月查看其他月份的記帳紀錄
- [x] 頁面有新增記帳按鈕
- [x] 點擊按鈕跳出 Alert Dialog
- [x] Dialog 可輸入金額
- [x] Dialog 可選取項目（吃飯、玩樂、薪水、租屋費、其他）
- [x] 選「其他」時可自定義名稱
- [x] 新增後資料存入 Room 資料庫
- [x] 重開 APP 後仍可看到之前新增的項目

---

## 邊界條件

- **金額**：正整數，不允許空白或負數
- **項目選取**：單選，預設選第一項（吃飯）
- **其他名稱**：選「其他」時必填，不可空白送出
- **月份切換**：最早可查詢到有資料的月份（不限制上限）
- **空清單**：當月無資料時顯示空狀態提示

---

## 不在範圍內（Out of Scope）

- 雲端同步 / 多裝置共享
- 記帳項目的編輯與刪除
- 統計圖表分析（屬 ChartScreen 範疇）
- 使用者登入 / 配對邏輯
