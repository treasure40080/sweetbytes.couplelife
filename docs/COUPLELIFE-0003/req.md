# COUPLELIFE-0003 統計圖表頁面開發 需求規格

## 功能描述

建立「統計」頁面，以圓餅圖視覺化呈現當月（或選定月份）的記帳資料分佈。
分兩個維度顯示：
1. **收入 vs 支出** 比例
2. **各分類**（吃飯、玩樂、薪水、租屋費、其他自定義）佔比

不同項目以不同顏色區分，並顯示各項目的比例數值。

---

## 使用者情境

**誰**：APP 使用者  
**情況**：進入統計頁籤  
**發生什麼**：想了解本月收支與各分類的花費分佈  
**期望結果**：看到清楚的圓餅圖，標示各分類顏色與比例

---

## 驗收條件

- [x] 進入頁面預設顯示當月資料
- [x] 可切換年/月查看其他月份
- [x] 顯示「收入 vs 支出」圓餅圖，兩種不同顏色
- [x] 顯示「各分類佔比」圓餅圖，每個分類不同顏色
- [x] 圖表旁或下方顯示圖例（顏色 + 分類名稱 + 百分比）
- [x] 無資料時顯示空狀態提示
- [x] 資料來源為 Room（與記帳流水頁面共用 EntryRepository）

---

## 邊界條件

- **單一項目**：若只有一種分類，圓餅圖顯示 100%
- **空資料**：當月無記帳時顯示「本月尚無記帳資料」，不顯示圖表
- **自定義名稱**：其他（自定義）各算獨立分類，不合併
- **百分比計算**：四捨五入至整數，合計不強制 100%

---

## 不在範圍內（Out of Scope）

- 折線圖 / 長條圖等其他圖表類型
- 跨月份趨勢比較
- 圖表動畫效果
- 匯出 / 分享功能
